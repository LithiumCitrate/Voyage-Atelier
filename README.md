# Voyage Atelier

Voyage Atelier is a travel itinerary planning website built with plain HTML, CSS, and JavaScript.

## Features

- Edit trip basics: name, destination, date range, pace, budget, currency, and notes
- Add itinerary items by day with time, place, type, cost, booking status, and notes
- Pick locations directly on the map before adding an itinerary item
- Draw route lines between mapped itinerary items with different colors for different days
- Review the trip through a day-by-day timeline and summary cards
- Import and export itinerary data as JSON files
- Keep browser storage empty by default, so the site does not retain personal trip data

## Run

Open `index.html` in a browser.

## Stack

No build step and no dependency install required.
